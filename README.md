# 🤖 AI-Powered DSA Telegram Bot

## 🚀 Overview
This project is an AI-powered automation bot that sends daily Data Structures and Algorithms (DSA) questions to users via Telegram.

## 🔧 Tech Stack
- Google Apps Script
- Telegram Bot API
- OpenRouter API
- DeepSeek AI Model

## ⚙️ Features
- Sends 3 DSA questions daily (Easy, Medium, Hard)
- Fully automated using time triggers
- Serverless (no hosting required)
- Supports multiple users

## 🧠 How It Works
1. Google Apps Script runs daily via trigger
2. Calls OpenRouter API with DeepSeek model
3. Generates DSA questions
4. Sends them to Telegram bot

2. Get Chat ID
3. Create OpenRouter API key
4. Paste keys in script
5. Set trigger in Apps Script
